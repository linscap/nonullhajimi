This is a proxy server for the Google Gemini API, designed to enhance stability and provide robust retry mechanisms.

This endpoint itself is not a user-facing application. It's a middleware that you should use in your own applications.

To use it, simply replace the official Gemini API base URL (https://generativelanguage.googleapis.com) with the URL of this deployment in your client code.

For example, a request to /v1beta/models/gemini-pro:generateContent will be proxied to the official API.
